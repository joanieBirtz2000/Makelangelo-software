Rapport des Flags de l'Action GitHub - IFT3913

Date : 2024-10-08

Auteurs: Joanie Birtz (20244554) & Janic Fournel (20201900)

Cette page documente les flags ajoutés à l'Action GitHub pour les tests. Chaque flag est décrit avec son objectif, sa motivation, et son impact attendu sur la qualité, la performance, et l'observabilité.

Nom : Run Tests with Memory Dump on Failure

Flag : -XX:+HeapDumpOnOutOfMemoryError

Motivation : Ce flag est activé pour améliorer l'observabilité. En cas de problème de mémoire (OutOfMemoryError), il permet de générer un dump de mémoire qui aide les développeurs à diagnostiquer les causes des fuites de mémoire ou des consommations excessives. Cela contribue à la stabilité en facilitant l’identification des problèmes de mémoire.

Nom : Run Tests with Optimized Garbage Collection

Flag : -XX:+UseG1GC

Motivation : Le choix du collecteur de déchets G1 (Garbage-First) vise à optimiser les performances de l'application. Ce collecteur est conçu pour réduire les pauses et améliorer le temps de réponse, ce qui est particulièrement utile pour les applications sensibles aux performances. Cela contribue à maintenir la réactivité de l’application.

Nom : Run Tests with Custom Stack Size

Flag : -Xss512k

Motivation : La taille de pile personnalisée de 512 Ko a été choisie pour identifier les problèmes liés à la récursivité excessive ou aux débordements de pile dans des environnements à mémoire limitée. Ce test aide à garantir que l'application peut fonctionner efficacement avec une utilisation optimale de la pile, contribuant ainsi à la robustesse du code.
